# MODERN-WEATHER-DASHBOARD
A beautiful, responsive weather application with neumorphic design and real-time data. It shows all data regarding weather updates and an unique feature which enables user to also plan future vacations and check weather on the particular day. It also contains a customized cursor which makes the website look dynamic.
